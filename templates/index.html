User Guide

Everything you need to know about using this system

🎯 Getting Started

1. Process a Consultation

Start by uploading an audio recording or pasting a text transcript of a client consultation. The AI will automatically extract:

  • Client's care level needs (Independent, Assisted, Memory Care)
  • Budget range
  • Timeline (immediate, near-term, flexible)
  • Location preference
  • Special requirements (pets, couples, enhanced services)

2. Review Recommendations

The system uses an 8-dimension AI ranking algorithm to provide the top 5 community recommendations:

  • Rule-Based Rankings: Business value, cost efficiency, distance, budget fit, couple suitability
  • AI-Powered Rankings: Availability match, amenity fit, holistic compatibility

Each recommendation includes detailed reasoning and key metrics.

3. CRM Integration

Results are automatically pushed to Google Sheets CRM (if configured) with three sheets:

  • Client Consultations: Overview of each consultation
  • Recommendations Detail: All 5 recommendations with full details
  • Performance Analytics: Processing metrics and costs

🗄️ Community Database Management

Adding a Community

  1. Go to "Community Database" tab
  2. Click "Add Community" button
  3. Fill in the required fields (marked with *)
  4. Click "Save Community"

Editing a Community

  1. Find the community in the table
  2. Click the edit (pencil) icon
  3. Update the fields
  4. Click "Update Community"

Deleting a Community

  1. Find the community in the table
  2. Click the delete (trash) icon
  3. Confirm the deletion

⚠️ Important: Changes to the database affect future recommendations immediately.

📊 Understanding Results

Recommendation Cards

Each recommendation shows:

  • Rank: 1-5, with 1 being the best match
  • Combined Score: Lower is better (weighted Borda count)
  • Monthly Fee: Base monthly cost
  • Distance: Miles from preferred location
  • Availability: Waitlist status
  • AI Reasoning: Why this community is recommended

Performance Metrics

  • Processing Time: Total time to generate recommendations
  • Token Usage: API tokens consumed (audio input, text input, output)
  • Cost: Estimated API cost based on Gemini 2.5 Flash pricing

💡 Tips & Best Practices

Audio Recording Tips

  • Ensure clear audio quality
  • Speak clearly and include all relevant details
  • Mention budget, timeline, and location explicitly
  • Supported formats: M4A, MP3, WAV, OGG

Text Consultation Tips

  • Include conversation format (Agent: / Client:)
  • Be specific about requirements
  • Mention ZIP code or city name
  • State timeline clearly (immediate = <1 month, near-term = 1-3 months)

Database Management Tips

  • Keep monthly fees up-to-date for accurate recommendations
  • Update waitlist status regularly
  • Use consistent ZIP code formatting (5 digits)
  • Fill in all important fields for better AI matching

⚙️ System Requirements

Works best on modern browsers:

Required environment variables:

❓ Troubleshooting

Common Issues

Audio processing fails

Solution: Check file format (must be M4A, MP3, WAV, or OGG) and file size (max 50MB). Ensure audio is clear and not corrupted.

"Not ranked by AI" appears in results

Solution: This means the AI ranking timed out. The system uses a fallback algorithm. Results are still valid but less detailed. This usually happens due to API rate limits.

CRM push fails

Solution: Verify Google Sheets credentials are configured correctly in .env file and that the spreadsheet is shared with the service account.

Database changes don't reflect

Solution: Refresh the page after making database changes. The system reloads the data automatically.
